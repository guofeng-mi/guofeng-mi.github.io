Linux常用命令

发表于 2020-12-15

常用的文件、目录操作命令

  • pwd : 查看用户的当前目录
  • cd : 命令来切换目录
  • . : 当前目录
  • .. : 当前目录的上一级目录（父目录）
  • - : 用 cd 命令切换目录前所在的目录
  • ~ : 用户主目录的绝对路径名
  • ls : 显示文件或目录信息
  • mkdir : 当前目录下创建一个空目录
  • rmdir : 要求目录为空
  • touch : 生成一个空文件或更改文件的时间
  • cp : 复制文件或目录
  • mv : 移动文件或目录、文件或目录改名
  • rm : 删除文件或目录
  • ln : 建立链接文件
  • find : 查找文件
  • file /stat: 查看文件类型或文件属性信息
  • cat : 查看文本文件内容
  • more : 可以分页看
  • less : 不仅可以分页，还可以方便地搜索，回翻等操作
  • tail -10: 查看文件的尾部的10行
  • head -20: 查看文件的头部20行
  • echo : 把内容重定向到指定的文件中 ，有则打开，无则创建

文件打包和压缩命令

压缩命令

  • gzip filename
  • bzip2 filename
  • tar -czvf filename

解压命令

  • gzip -d filename.gz
  • bzip2 -d filename.bz2
  • tar -xzvf filename.tar.gz

vim程序编辑器

vim是vi文本编辑器的进阶版，在所有linux系统上均可使用

vi分为三种模式：一般模式，编辑模式和指令列命令模式

vi hello.c : 进入 hello.c 文件，此时为一般模式，若 hello.c 不存在则自动新建

一般模式下执行vi定义的键盘上转义字符操作，

  • del : 删除光标上字符（mac键盘为 option + 删除键）
  • dd : 删除一整列，退格键及上下左右键移动光标
  • i : 进入插入光标前编辑模式
  • a 进入插入光标后编辑模式
  • r 进入改写编辑模式
  • q! 不保存退出

编辑模式下键盘上所有按键除ESC外都可输入字符，如上下左右键分别代表ABCD大写字母，按ESC退出编辑；

在一般模式下按 : 进入指令列模式，最下面一行为指令列，wq 保存并退出。
