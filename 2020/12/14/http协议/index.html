发表于 2020-12-14 更新于 2020-12-15

五层网络协议

  1. 应用层：客户端（微信，QQ）
  2. 传输层：交换机（TCP、UDP）
  3. 网络层：IP协议
  4. 数据链路层：ATM、FDBI
  5. 物理层：电流、编码、频率
graph LR
    主机A --> 互联网 --> 主机B
  • 应用层: 浏览器（HTTP、FTP[文件传输协议]、DNS[域名解析]、SMTP[邮箱]、Telnet[Linux端口建立连接]）
  • 传输层: TCP、UDP
    • TCP: 关注结果
    • UDP: 不关注结果
  • 网络层: IP协议（地址），ICMP、RIP、BGP
  • 数据链路层: 定义数据如何传输，PPP，SLIP
  • 物理层: OSI 物理层规范（传输环节具体的规范），ISO2110，IEEE802

七层网络协议

  1. 应用层：客户端（微信，QQ）
  2. 表示层: 加密，格式转换
  3. 会话层: 解除或者建立和其他节点的联系
  4. 传输层：交换机（TCP、UDP）
  5. 网络层：IP协议
  6. 数据链路层：ATM、FDBI
  7. 物理层：电流、编码、频率

HTTP(HyperText Transfer Protocol，超文本传输协议)

  • Request(发送)

    • 请求头
    • 数据体

  • Response(返回)

    • 返回头
    • 数据体
